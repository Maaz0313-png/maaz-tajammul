# Maaz Tajammul Portfolio

Personal portfolio website for **Maaz Tajammul** (Full Stack Laravel Developer), built as a single-page site in `index.html`.

## Website Sections

- Home / Hero
- About
- Skills
- Projects
- Experience
- Education
- Contact

## Tech Stack

- HTML (single-page structure)
- Tailwind CSS (compiled from `src/input.css` to `src/output.css`)
- Font Awesome icons

## Local Development

1. Install dependencies:

```bash
npm install
```

2. Build styles:

```bash
npm run build
```

3. Run Tailwind in watch mode:

```bash
npm run watch
```

4. Open `index.html` in your browser.

## Repository

- Main page: `index.html`
- Tailwind source: `src/input.css`
- Tailwind output: `src/output.css`
